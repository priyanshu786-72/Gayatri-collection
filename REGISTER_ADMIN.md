# Register Admin User - Step by Step

## Step 1: Register
1. The login page should be open in your browser
2. Click **"Sign up"** link at the bottom
3. Fill in:
   - **Name:** Admin
   - **Email:** admin@test.com
   - **Password:** (choose any password, e.g., admin123)
   - **Phone:** (optional)
   - **Role:** Select "Customer" (we'll change it)
4. Click **"Sign Up"**

## Step 2: Make Admin
After you register, come back here and I'll make you admin!

Or run this command yourself:
```powershell
cd Gayatri-shop/backend
node createAdmin.js admin@test.com
```

## Step 3: Login Again
1. Go back to login page
2. Login with: admin@test.com and your password

## Step 4: Access Admin Panel
Go to: http://localhost:8000/admin.html
